# Student Management System

A Java-based Student Management System developed to manage student records.
The application allows users to add and view student information.

## Technologies Used

- Java

## Features

- Add student details
- View student records
- Store student information
- Menu-driven application

## Concepts Used

- Classes and Objects
- OOP Concepts
- ArrayList
- Scanner Class

## Project Purpose

This project helps to understand Java programming fundamentals and object-oriented programming concepts.

## How to Run

1. Install Java JDK
2. Compile the file:

javac StudentManagementSystem.java

3. Run the program:

java StudentManagementSystem
